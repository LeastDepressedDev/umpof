# Фронт

На нодах реализуется пайплайн. 
Ноды берутся из паков.(Паки описаны позже)
Фронт запрашивает доступные паки с серверной части.

# Бэкенд

Фронт через буферку отправляет запрос ядру -> ядро создаёт сессию -> 
Режимы компилируемый/интропретируемый:
- При компилируемом режиме на сервер отправляется .iso с докером, он запускает его и работает уже с ним
- При интропретируемом режиме сессия попроцессно отправляет запросы серверу, на исполнение
